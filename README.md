# Sprint4_CarePlus_Saude_IoT

## Objetivo

Sistema inteligente de monitoramento de pacientes utilizando ESP32 e sensores biomédicos.

## Componentes

- ESP32 DevKit V1
- DHT22
- Potenciômetro BPM
- Potenciômetro SpO₂

## Funcionalidades

- Monitoramento de temperatura
- Monitoramento BPM
- Monitoramento SpO₂
- Classificação automática
- Envio MQTT
- Dashboard Node-RED
- Histórico de dados

## Arquitetura

Sensores → ESP32 → MQTT → Node-RED → Dashboard

## Tecnologias

- ESP32
- MQTT
- Node-RED
- Wokwi
- GitHub
